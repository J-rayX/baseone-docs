# Errors

The common errors you could have when using BaseOne APIs.

BaseOne APIs are RESTful, therefore, they use conventional HTTP response codes to demonstrate the success or failure of API requests.

## Error Codes

Some common errors codes and their causes are:

| Error Code   | Reason for Error                                                                                                        |     |
| :----------- | :---------------------------------------------------------------------------------------------------------------------- | :-- |
| `200`, `201` | Request was successful.                                                                                                 |     |
| `400`        | Request was not achieved due to a validation or client side error.                                                      |     |
| `404`        | Request was not achieved because the resource requested for does not exist.                                             |     |
| `500`, `501` | Request not achieved due to an error on our side. This is not your fault. It's us. Kindly report if you get this error. |     |
|              |                                                                                                                         |     |
